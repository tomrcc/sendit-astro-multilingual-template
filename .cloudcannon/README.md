# Let's see CloudCannon in action ⚡

This is **Sendit**, a fully configured Astro marketing template. Use it to explore how CloudCannon works — edit pages visually, write blog posts, and manage site data.

This *Site* is your sandbox, so feel free to experiment.

## Try the *Visual Editor*

Open a page and click directly on any text, image, or component to edit it in place. Try changing a headline, swapping a call-to-action, or rearranging page sections.

[Open Pages](cloudcannon:collections/pages)

## Write in the *Content Editor*

Create or edit a blog post using the *Content Editor* — a clean, Markdown-based writing experience with a familiar formatting toolbar.

[Open Blog Posts](cloudcannon:collections/posts)

## Manage structured data

Update your navigation links, footer content, or theme colors in the *Data Editor*. Changes flow through to every page automatically.

[Open Site Data](cloudcannon:collections/data)

## How editing works

CloudCannon provides three editing interfaces, each designed for different tasks:

- **Visual Editor** — Click directly on your live preview to edit content in context. Ideal for marketing pages and component layouts.
- **Content Editor** — A rich text editor for long-form content like blog posts and articles.
- **Data Editor** — Edit structured fields like titles, metadata, and settings in a form-based interface.

Every edit saves back to your Git repository, keeping your development workflow intact.
